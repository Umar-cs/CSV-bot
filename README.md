An intelligent chatbot that allows you to interact with CSV data using natural language queries.
Built with PandasAI, this project combines LLM-powered reasoning with efficient caching to reduce latency and API costs.


🚀 Features

💬 Ask questions about CSV data in plain English

📊 Automatic data analysis and visualization

⚡ Smart caching for faster repeated queries

🧠 LLM-powered reasoning over structured data

🔌 Works with OpenAI or local LLMs

📁 Supports large CSV files

Flow:

Load CSV into a Pandas DataFrame

User asks a question (e.g., “average salary?”)

PandasAI converts it into executable Python code

Result is generated using Pandas

Response is cached for future queries
